May 2023

In Progress.

  • Moved build process over to C99 from rust.

  • Automated index log file addition into build process.

The aim is to create a minimal, self-contained, zero-dependency binary that builds the website from html templates. There's still quite a few improvements to make, but the first steps are in place. 

time make
gcc -o main src/main.o src/file_utils.o src/dir_utils.o -I.
real	0m0.014s
user	0m0.008s
sys	0m0.006s

time ./main
The build was successful
real	0m0.001s
user	0m0.001s
sys	0m0.000s

Some short term goals are:

  • A basic set of styling applied to the base html tags and to remove the use of classes, like in normalize.css.

  • Automate the 'Last Edited' footer tag, per page.

  • Include additional 'meta' information about the file within the footer, per page.
