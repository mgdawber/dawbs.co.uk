Hi, my name is Matt. I work as a programmer, and sometimes draw in my free time. I'm using this space as a way to document my drawings and animations.

Website

This document functions as a personal memex, where I collect and store information that I have learned, experienced, or find otherwise interesting. It is a living document that grows and changes overtime. It is heavily inspired by 100 rabbits.

This website consists of static HTML pages generated by a C binary. It loads no JavaScript or any external content, and is designed to be as minimal as possible. It can be viewed offline without the need for extra tooling or local servers.

Page load should also be kept as small as possible (< 100Kb). As such, images and media are generally dithered.
